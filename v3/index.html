IR Data Compiler

IR Data Compiler is a tool I developed for a friend that needed a simple way to compile many small files into one large file for data handling. It was made to specifically combine CSV files outputed by Bruker OPUS spectroscopy software, however it will have many other use cases depending on the format of data. Written entirely in java, IRDC provides an intutive window to define directories of files, and even offers normalization of data.
